myFlix

What is this?

This application is a REST API created through REACT which provides access to a database with information about different movies, directors, and genres.

The user can also sign up and update their information within the site to save information about the movies that interest them the most.

Endpoint Documentation: CRUD

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title.
  • Return data about a director (bio, birth year) by name.
  • Allow new users to register.
  • Allow users to update their user info (username).
  • Allow users to add a movie to their list of favorites.
  • Allow users to remove a movie from their list of favorites.
  • Allow existing users to deregister.
Request URL HTTP Method Response
Return a list of ALL movies to user /movies GET a JSON object holding data about all movies
Return data about a single movie by title to the user /movies/:title GET a JSON object holding data about a single movie containing the data above
Return data about a genre by name/title /movies/genre/:genreName GET a JSON object holding data about genre name and its description
Return data about a director (bio, birth year, death year) by name /movies/director/:directorName GET a JSON object holding data about director such as name, bio, date of birth
Allow new users to register /users POST a JSON object holding information about a newly created user account
Allow users to update their user info (username) /users/:UserName PATCH a JSON object holding updated information of a user account
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added—more on this later) /users/:Username/movies/:MovieID POST a JSON object containing information of a user account with movieID added
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed—more on this later) /users/:Username/movies/:MovieID DELETE a JSON object holding updated data with a requested movie deleted from the favorites list
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) /users/:UserName DELETE a text message indicating a user account was deleted
